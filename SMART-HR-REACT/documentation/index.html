Installation Guide

After unzipping Template Files, you will find our product Smarthr ->Reactjs(Source code)

Mandatory are npm and node are needed to run the reactjs project

Once unzipped the files,run the command:

npm install (or) npm i

For running a project , run the command:

npm start

If you want to run the project and make the build in the production mode then run the following command in the root directory, otherwise the project will continue to run in the development mode. 

npm run build

To install any plugins in your application, you have to run the following command 

npm install plugin_name --save

How can I switch to the Dark Mode?

If you use Dynamic Theme option. You will see a Light mode and Dark mode radio button in the Theme Settings. Choose any one of the option to change website theme.

If you need to set static Theme to your template please data-theme="dark" in <html data-theme="dark"></html> at index.html file in public folder.

How can I switch to the RTL?

  • Add Bootstarp RTL

    Please remove marked style in src/index.tsx file and replace it with below style 

    "import "../node_modules/bootstrap/dist/css/bootstrap.rtl.min.css";"
  • Add Class in Body tag in public/index.html file
    class="layout-mode-rtl"

How Can I Use Theme Layouts?

Please do the below change in the html element at public/index.html file. ex: <html data-theme="default">

data-layout="default"
data-layout="mini"
data-layout="horizontal"
data-layout="horizontal-single"
data-layout="detached"
data-layout="twocolumn"
data-layout="without-header"
data-layout="horizontal-overlay"
data-layout="horizontal-sidemenu"
data-layout="stacked"
data-layout="modern"
data-layout="transparent"
